# Islam in Nepal - Admin Panel

A complete website about Islam in Nepal with an admin panel for content management.

## Features

- **Public Website**: Information about Islam in Nepal with responsive design
- **Admin Panel**: Secure content management for allegations and clarifications
- **Authentication**: Secure login with session management
- **CRUD Operations**: Create, read, update, delete allegations and clarifications
- **Search & Pagination**: Easy content management with search and pagination
- **Responsive Design**: Works on all devices
